## 1. SETUP

### 1.0. Creating your first React App

react 를 설치하려면 `node.js 10.16 이상, npm 5.6` 이상의 버전이 설치 되어 있어야 한다.

react 설치 명령어

```
npx create-react-app `{폴더 이름}`
```
